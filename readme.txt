Google search in a blog widget. 

== Description ==

A simple Wordpress sidebar widget for searching in blog`s domain by Google.
Uses JavaScript.

== Installation ==

1. Change your blog`s domain name in wp_config.php
2. Upload wp_google.php to plugins directory
3. Activate the plugin in 'Plugins' menu of wp-admin
4. Add a widget to sidebar. Enjoy!
